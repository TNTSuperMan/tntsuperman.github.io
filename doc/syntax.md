# Whxute文章ファイル

## /[文章]
p要素で文を出力します。  
連続でやったらつながります。
## :[要素名]:[内容]
要素を出力します。  
:p:やあ と入れたら
```
<p>やあ</p>
```
という具合に
## =[属性名]=[属性内容]
:要素名:内容 のやつに属性を加えます。
## \\[テンプレ名]\\[内容]\\...
設定で指定したテンプレファイルの通りにテンプレよびだします
## &[内容]
HTMLをそのまま出力します  
JSのinnerHTML+=でやってるので物によってバグあるかも
## -[要素名]
## -[要素名]-[オプション]
:要素名:内容の複数行バージョン的な  
要素グループかな  
この中身はWhxute.jsのルールで構成します  
オプションでDIRECTだとそのままHTML出力します。
## +
-なんちゃらのグループを終わらせます。
